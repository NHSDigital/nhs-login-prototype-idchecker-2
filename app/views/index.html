Audit view

Assurance audit is a process for ID checkers to audit themselves. It is carried out by trained supervisors only.

Audit view is not accessed via ID checker dashboard.

Spluk randomly selects 10% of the approved checks and generates a list. Supervisors log in to Splunk and copy this list and create links on a spreadsheet. They then manually assign these to themselves and go over individual checks link by link.
